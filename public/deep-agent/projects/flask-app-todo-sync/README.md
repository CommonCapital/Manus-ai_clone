# Flask App Todo Sync

## Overview
A simple Flask app with a landing page. This project was created to demonstrate the project organization skill and Todo Sync Protocol.

## Quick Start
```bash
# Installation
pip install -r requirements.txt

# Run
python src/app.py
```
