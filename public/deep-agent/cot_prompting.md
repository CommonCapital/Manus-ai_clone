# Chain-of-Thought (CoT) Prompting

## What is Chain-of-Thought Prompting?
Chain-of-Thought (CoT) prompting is a technique for getting large language models (LLMs) to reason step‑by‑step before arriving at an answer. Instead of asking the model for a direct answer, you ask it to **explain its reasoning** in natural language. This often leads to more accurate and interpretable results, especially on complex problems that require multi‑step reasoning.

## Why Use CoT Prompting?
- **Improved accuracy** on tasks like arithmetic, logic puzzles, and symbolic reasoning.
- **Transparency** – you can see the model's intermediate thoughts.
- **Error diagnosis** – if the final answer is wrong, the chain can reveal where the reasoning went off track.
- **Better alignment** – the model follows a more human‑like reasoning process.

## Basic CoT Prompt Structure
```
You are a helpful assistant.

Question: <INSERT QUESTION>

Answer the question step by step, showing your reasoning.
```
**Example**:
```
You are a helpful assistant.

Question: If a train travels 60 miles in 1.5 hours, what is its average speed in miles per hour?

Answer the question step by step, showing your reasoning.
```
**Model output**:
```
Step 1: Identify the formula for average speed: speed = distance / time.
Step 2: Plug in the values: distance = 60 miles, time = 1.5 hours.
Step 3: Compute the division: 60 / 1.5 = 40.
Step 4: The average speed is 40 miles per hour.
Answer: 40 mph.
```

## Advanced Techniques
### 1. Few‑Shot CoT
Provide a few examples of the reasoning process so the model can mimic the pattern.
```
Q: 12 + 23 = ?
A: Step 1: Add the tens: 10 + 20 = 30.
   Step 2: Add the units: 2 + 3 = 5.
   Step 3: Combine: 30 + 5 = 35.
   Answer: 35

Q: 7 * 8 = ?
A: ...
```
### 2. Self‑Consistency
Generate multiple CoT answers (e.g., using temperature > 0) and select the most common final answer.
### 3. Zero‑Shot CoT
Even without examples, simply appending *"Let's think step by step."* often triggers chain‑of‑thought behavior.
```
Question: ...
Answer: Let's think step by step.
```

## Common Pitfalls
- **Too verbose** – overly long chains can drift off topic.
- **Hallucination** – the model may invent facts while reasoning; verify each step.
- **Prompt length** – large prompts may hit token limits; keep examples concise.

## Sample Prompt Library
| Prompt Variant | Description |
|----------------|-------------|
| Zero‑Shot CoT | Append "Let's think step by step." to any question. |
| Few‑Shot CoT | Provide 2‑3 solved examples before the target question. |
| Structured CoT | Use numbered steps (`Step 1:`, `Step 2:`) for clarity. |
| Self‑Consistency | Sample multiple CoT completions and vote on the final answer. |
